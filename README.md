# Welcome to the Learning Journey "SAP UI5 Development"
[REUSE status] (https://github.com/anukarnam/Sapui5development.git)

Welcome to this repository for the learning journey "SAP UI5 Development" published on [SAP Learning](https://learning.sap.com/learning-journey/). 

## Requirements
You can complete each of the exercises using your SAP BTP Free Tier account.

## Download the Installation

To get started with the first exercise, simply run the following commands in your terminal:

```sh
git clone https://github.com/anukarnam/Sapui5development.git
```

I recommend to follow the instructions in the [learning journey](https://github.com/anukarnam/Sapui5development.git) to get started.


If you want to start or continue from a specific unit or exercise, get the name of its branch from the tutorial, clone this repository and switch to the desired branch:

```sh
git checkout <anukarnam>
```
